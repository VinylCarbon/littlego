Fuego

Fuego is the Go program that you are playing against. It is licensed under the LGPL, Version 3.

Boost

Boost is the software library that makes Fuego (and with it this app) possible in the first place. It is licensed under the Boost Software License, Version 1.0.

Other software

MBProgressHUD is the code snippet responsible for displaying a progress meter in this app. The code has been made available under the MIT license.

Cocoa Lumberjack is a very nice logging framework that I use for developing %bundleName%. The Cocoa Lumberjack source code has been made available under the BSD 2-Clause license.

ZipKit is an Objective-C framework for reading and writing Zip archives in Mac OS X and iOS apps. ZipKit was developed by Karl Moskowski and has been made available under a BSD-style license.

QuincyKit is an Objective-C component for sending crash reports to a predefined server, both for Mac OS X and iOS apps. Internally QuincyKit uses two additional pieces of software: The PLCrashReporter

framework, and the protobuf-c library. QuincyKit and PLCrashReporter have been released under the MIT license, while protobuf-c is available under the Apache License v2.0.

Icons

Most of the icons are by Glyphish, from the Pro icon set. If you need artwork for a project of your own but don't want to spend hours fumbling around with the Gimp, you should really check out this guy's work. There's also a free icon set, by the way.

Little Go still lacks stylish images of black and white Go stones. Please contact me if you are interested in contributing your art to this project.
