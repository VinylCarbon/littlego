Littlego

Little Go. An iOS application that lets you play the game of Go on the iPhone or iPad.

Download .zip Download .tar.gz View on GitHub

Little Go is an iOS application that lets you play the game of Go on the iPhone or iPad. You can play against another human, or against the computer. The minimum requirement for running this version of Little Go is iOS 5.0.

For a detailed description of Little Go's features, see the MANUAL document (also available in-game from the "Help" tab).

If you are interested in Little Go as a developer, you can either clone the source code repository on GitHub, or download a release snapshot of the source code for one of the release tags. Release snapshots are also available from GitHub. Either way, you should read the quick-start guide in README.developer to get up and running.

Changes in this release

This is the Little Go feature release 0.10.0. A large number of changes have been made, only the most important ones are listed here:

  • Improve graphic design (wooden background, new artwork for stones, new layout on iPad)
  • View board positions for moves played earlier during the game
  • Replace "undo" with "discard move"
  • A game that has ended can now be resumed
  • Redesign of "New game" and "Game info" screens
  • 3 new user preferences
  • Add support for collecting logging data
  • Drop support for iOS 4.3, the minimum required version is now iOS 5.0
  • Add iOS 6 compatibility
  • Add iPhone 5 support

The ChangeLog document has more details.

Documentation

  • The source code is documented with Doxygen. The project is set up to generate the documentation with a a premade Doxyfile configuration file and an Xcode target that expects Doxygen and GraphViz to be located in /opt/local/bin (the standard MacPorts path).
  • A class diagram in the doc folder (a .jpg file) provides an overview of Little Go's core classes. The diagram was created with the free community edition of the UML modeling application Astah. The model itself is in the file littlego.asta.
  • The end user documentation, which is available from within the game, is in doc/MANUAL.
